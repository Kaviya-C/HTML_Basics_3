HTML CANVAS

Html canvas used to draw the graphics,canvas element is only a container for graphics
we must use javascript to actually draw the graphics.Canvas has several methods for
drawing paths,boxes,circles,text and adding images
by default canva has no border and no content it is a rectangulat area on the html page



Key methods in canvas element

  1. DRAWING PATH
    1. beginPath() ==starts new path or resets the current path
    2. moveTo(x,y)==move the current drawing path to the specified coordinates
    3. lineTo(x,y)==Draw a straight line from the current point to the specified coordinates
    4. arc(x,y,radius,startAngle,endAngle,anticlockwise) == draws an arc or a portion of a circle
    5. rect(x,y,width,height)==draw a rectangle
    6. quadraticCurveTo(cpx,cpy,x,y)==draws a quadratic bezier curve from the current point to the endpoint
    7. bezierCurveTo(cp1x,cp1y,cp2x,cp2y,x,y)==draws a cubic bezier curve from the current point to the endpoint
  2. Drawing styles
    1. stroke()==strokes the current path with the current stroke style
    2. fill()==fill the current path with the current fill style
    3. strokeStyle==sets the color ,gradient,pattern used for strokes
    4. fillStyle==sets the color,gradient,pattern used for fill
  3. Line and shape styles
    1. lineWidth=="sets the width of lines"
    2. lineCap==sets the style used to draw the ends of lines
    3. lineJoin== sets the style used to determine the shape at the corners where lines meet
    4. miterlimit==sets the maximum miter length for lineJoin='miter'
  4. Text
    1. font==set the font properties for text rendering
    2. textAlign==sets the horizontal alignment of text
    3. textBaseline==sets the baseline alignment
    4. fillText(text,x,y)==fill the given text at the specified (x,y) position
    5. strokeText(text,x,y)==strokes the given text at the specified (x,y)position
  5. Image Drawing
    1. drawImage(Image,x,y)== draw an image video or canvas on the canvas
    2. createPattern(image,repetition)== creates a pattern from an image ,video or canvas
  6. Transformations
    1. translate(x,y)==move the canvas and its origin to specified(x,y) position
    2. rotate(angel)== rotate the canvas around the current origin by the specified angle (in radians)
    3. scale(scaleX,scaleY)==scales the canvas by the specified factors in the x and y directions
    4. transform(a,b,c,d,e,f)==applies teh custom 2D transformation matrix to the current context
    5. setTransform(a,b,c,d,e,f)==resets teh current transformatiion matrix with a custom matrix
  7. Clipping
    1. clip()==clips the current path using the current path or a given PATH
    2. resetClip()==resets the clipping region to the full canvas
